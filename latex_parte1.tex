\frame{\frametitle{\LaTeX}
\begin{itemizespc}
\item O \LaTeX\ (Lamport Tex) é um processador de textos de alta qualidade tipográfica conforme ``marcações'' encontradas no texto
\item As ``marcações'' indicam como o texto deve ser formatado no que se refere a negrito, sublinhado, parágrafo, seções etc.
\item \LaTeX\ é software livre, sendo distribuído com a licença {\it LaTeX Project Public License} (LPPL)
\end{itemizespc}
}

\frame{\frametitle{\TeX}
\begin{itemizespc}
\item O \LaTeX\ faz uso dos recursos tipográficos de outro programa conhecido como \TeX, que é um sistema tipográfico
\item \TeX\ foi concebido com o objetivo de:
\begin{itemize}
\item permitir a preparação de documentos com alta qualidade tipográfica utilizando o mínimo de esforço
\item fornecer um sistema que produza documentos idênticos em todos os computadores, a qualquer momento
\end{itemize}
\item \TeX\ também é software livre, cuja versão mais atual é $3.14159265$ (01/12/2014)
\end{itemizespc}

\frame{\frametitle{\LaTeX vs \TeX}
\begin{itemizespc}
\item \LaTeX\ é um conjunto padrão de macros implementados utilizando o \TeX
\begin{itemize}
\item permite um aumento da produtividade com o uso desse processador
\end{itemize}
\item Resumidamente:
\begin{itemize}
\item \LaTeX\ está mais relacionado com a manipulação do conteúdo (texto) juntamente com as marcações, tarefa feita pelo escritor
\item \TeX\ está mais relacionado com as funcionalidades que fornecem o layout, tarefa feita pelo tipógrafo
\end{itemize}
\end{itemizespc}
}

\frame{\frametitle{Um pouco de história}
\begin{itemizespc}
\item Nos anos 70, um professor da Universidade de Stanford, chamado Donald E. Knuth, estava escrevendo um livro sobre programação
\end{itemizespc}
\begin{figure}
\includegraphics[scale=0.4]{figs/knuth.png}
\end{figure}
}

\frame{\frametitle{Um pouco de história}
\begin{itemizespc}
\item O livro em questão era ``A arte de programação do computador'', que após finalizado, foi enviado à editora para impressão
\item A editora contratada enviou uma versão impressa do livro para Knuth, que ficou insatisfeito com sua baixa qualidade
\item Isso motivou Knuth a implementar um programa de editoração, o \TeX!
\end{itemizespc}
}

\frame{\frametitle{Um pouco de história}
\begin{itemizespc}
\item Os desafios de Knuth eram grandes:
\begin{itemize}
\item modelar as fontes (fonte padrão é denominada {\it Computer Modern})
\item desenvolver todos os ambientes existentes em textos científicos
\end{itemize}
\item O primeiro protótipo do \TeX\ foi lançado em 1978, enquanto que a primeira versão estável, em 1982
\end{itemizespc}
}

\frame{\frametitle{Compilando um arquivo \LaTeX}
\begin{itemizespc}
\item No Terminal do Linux, digite:
\begin{flushleft}
$\$ \quad latex <arquivo>.tex$ 
\end{flushleft}
\item Um arquivo .dvi (com o mesmo nome) será gerado
\item Para convertê-lo para PDF, digite:
\begin{flushleft}
$\$ \quad dvips -o <arquivo>.dvi <arquivo>$ \\
$\$ \quad ps2pdf <arquivo>.ps$ 
\end{flushleft}
\end{itemizespc}

\frame{\frametitle{Pacotes}
\begin{itemizespc}
\item O usuário verificará que textos mais complexos ficam difíceis de serem escritos com os recursos básicos do \LaTeX
\begin{itemize}
\item Solução: incluir pacotes no preâmbulo do arquivo \LaTeX\ 
\end{itemize}
\item Pacotes são um conjunto de arquivos que implementam uma determinada característica adicional para os documentos escritos em \LaTeX\ 
\item Alguns estão incluídos no código-fonte do \TeX, enquanto que outros requerem o download em repositórios públicos, como:
\begin{itemize}
\item \url{https://www.ctan.org/} - ``Comprehensive TEX Archive Network''
\end{itemize}
\end{itemizespc}
}

\frame{\frametitle{Pacotes}
\begin{itemizespc}
\item Os principais pacotes são:
\begin{itemize}
\item {\bf graphicx:} inserir gráficos, como imagens, fotografias etc
\item {\bf color:} utilizar cores no texto
\item {\bf babel:} traduzir automaticamente nomes que aparecem em inglês na estrutura do documento.
\begin{itemize}
{\it Section} $\rightarrow$ Seção, {\it Chapter} $\rightarrow$ Capítulo
\end{itemize}
\item {\bf fontenc:} permite que o \LaTeX\ compreenda a acentuação feita direto pelo teclado.
\item {\bf amsfonts:} utilização de estilos de letras para o ambiente matemático, como $\mathbb{N}$, $\mathbb{R}$ etc
\item {\bf amssymb:} utilização de símbolos matemáticos, como $\alpha$, $\beta$ etc.
\item {\bf float:} adequar o posicionamento de imagens
\end{itemize}
\end{itemizespc}
}

\frame{\frametitle{Pacotes}
\begin{itemizespc}
\item A inclusão de um pacote em um arquivo .tex requer o uso do comando
\begin{center}
$\backslash usepackage [< opcoes >]\{ < nome-do-pacote>\}$
\end{center}
\item Por exemplo:
\begin{itemize}
\item $\backslash usepackage \{ graphicx\}$
\end{itemize}
\item É importante consultar na documentação do pacote as opções disponíveis! 
\end{itemizespc}

\frame{\frametitle{Pacotes}
\begin{itemizespc}
\item Como verificar se é necessário a inclusão de um pacote no arquivo \LaTeX?
\begin{itemize}
\item No momento da compilação, o \LaTeX\ indicará algum comando que não foi reconhecido...
\begin{itemize}
\item o que requer a inclusão do pacote que implementa tal comando!
\end{itemize}
\end{itemize}
\item Se ao realizar a inclusão do pacote e o \LaTeX\ indicar sua ausência no sistema...
\begin{itemize}
\item será necessário o download do código-fonte do pacote (pode ser em formato .tex, .cls ou .sty)
\item mover o código-fonte do pacote para a pasta raiz do arquivo \LaTeX\ sendo editado
\item incluir o pacote no arquivo com o comando $\backslash usepackage$
\end{itemize}
\end{itemizespc}
}

\frame{\frametitle{Comandos básicos}
\begin{itemizespc}
\item Quebra de linha
\begin{itemize}
\item inserir ``$\backslash \backslash$'' no momento da mudança de linha
\end{itemize}
\item Parágrafo
\begin{itemize}
\item inserir ``$\backslash indent$'' ao iniciar o parágrafo
\item o comando ``$\backslash noindent$'' retira a identação de um parágrafo\\
\end{itemize}
\item Comentários no código: incluir $\%$ na linha a ser comentada
\vspace{0.5cm}
{\scriptsize 
\begin{columns}[T]
\begin{column}{.5\textwidth}
$\backslash noindent$ Meu primeiro documento em Latex!$\backslash \backslash$\\
$\backslash indent$ Muito interessante!
\end{column}
\hfill
\begin{column}{.5\textwidth}
\noindent Meu primeiro documento em Latex! \\
\indent $\quad$ Muito interessante!
\end{column}
\end{columns}}
\end{itemizespc}
}

\frame{\frametitle{Comandos básicos}
\begin{itemizespc}
\item {\bf Importante: } uma ou mais linhas em branco delimitam um parágrafo!
\item Comentários no código: incluir $\%$ na linha a ser comentada
\item Espaços em branco, tabulações e novas linhas são desprezados no momento da compilação
\end{itemizespc}
}
